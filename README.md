## 关于项目

这是一个开源的AI IDEA插件项目，如果使用阿里云百炼平台，运行项目时，首先将AiExecutor类中的ApiKey换成自己的key。

目前项目基于阿里云百炼平台的API： https://bailian.console.aliyun.com/#/model-market

后续会提供基于开源的框架和大模型的插件逻辑
